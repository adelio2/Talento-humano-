# Descripción del Proyecto

Este proyecto corresponde al desarrollo de un sistema de **Gestión de Permisos para Talento Humano**, orientado a facilitar el control, registro y administración de solicitudes de permisos de los empleados dentro de una organización.

El sistema busca optimizar procesos administrativos, mejorar la organización de la información y agilizar la aprobación de permisos laborales.
