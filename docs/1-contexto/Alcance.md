 Alcance del Proyecto

Definición del alcance

El proyecto consiste en el **desarrollo de un sistema de gestión de solicitudes de Talento Humano** para la empresa Nexa Servicios. Este sistema centralizará la administración de permisos laborales, vacaciones, certificaciones laborales e incapacidades médicas, mejorando la eficiencia operativa y la trazabilidad de los procesos.

## Objetivos del alcance

- Crear una plataforma centralizada para solicitudes de Talento Humano
- Eliminar los canales informales de comunicación (WhatsApp, correos dispersos)
- Garantizar la trazabilidad completa de cada solicitud
- Asignar responsables de manera clara y automática
- Mantener un historial detallado de todas las transacciones
