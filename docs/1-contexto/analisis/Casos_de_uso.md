## Casos de Uso

| Código | Caso de Uso | Actor Principal | Descripción |
|---|---|---|---|
| CU-01 | Registrar solicitud de permiso laboral | Empleado | Permite al empleado registrar solicitudes de permisos laborales dentro de la plataforma. |
| CU-02 | Solicitar vacaciones | Empleado | Permite al empleado realizar solicitudes de vacaciones mediante el sistema. |
| CU-03 | Registrar incapacidad médica | Empleado | Permite registrar incapacidades médicas adjuntando documentos de soporte. |
| CU-04 | Solicitar certificación laboral | Empleado | Permite solicitar certificaciones laborales desde la plataforma institucional. |
| CU-05 | Consultar estado de solicitud | Empleado | Permite visualizar el estado actualizado de las solicitudes realizadas. |
| CU-06 | Consultar historial de solicitudes | Empleado | Permite visualizar solicitudes anteriores y sus respuestas registradas. |
| CU-07 | Consultar responsable asignado | Empleado | Permite identificar el responsable encargado de revisar la solicitud. |
| CU-08 | Recibir notificaciones automáticas | Empleado | Permite informar al empleado sobre cambios en el estado de las solicitudes. |
| CU-09 | Visualizar solicitudes registradas | Talento Humano | Permite consultar todas las solicitudes registradas en el sistema. |
| CU-10 | Aprobar solicitudes | Talento Humano | Permite aprobar solicitudes realizadas por los empleados. |
| CU-11 | Rechazar solicitudes | Talento Humano | Permite rechazar solicitudes indicando observaciones o motivos. |
| CU-12 | Cambiar estado de solicitud | Talento Humano | Permite actualizar el estado de una solicitud durante el proceso. |
| CU-13 | Consultar historial y trazabilidad | Talento Humano | Permite consultar el historial de acciones realizadas sobre las solicitudes. |
| CU-14 | Generar reportes administrativos | Talento Humano | Permite generar reportes relacionados con solicitudes y tiempos de respuesta. |
| CU-15 | Gestionar responsables del proceso | Talento Humano | Permite asignar responsables para la revisión y seguimiento de solicitudes. |
| CU-16 | Validar información de solicitudes | Sistema | Permite verificar campos obligatorios y documentos adjuntos antes del registro. |
| CU-17 | Generar número de radicado | Sistema | Permite generar automáticamente un identificador único para cada solicitud. |
| CU-18 | Registrar trazabilidad de acciones | Sistema | Permite almacenar automáticamente cambios y acciones realizadas sobre las solicitudes. |
