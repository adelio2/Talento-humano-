## 7. Problemas Identificados

| Código | Problema identificado |
|--------|------------------------|
| P-01 | Falta de trazabilidad en el proceso de gestión de solicitudes realizadas por los empleados. |
| P-02 | Uso de múltiples canales informales (correos, mensajes y comunicación verbal) para realizar solicitudes internas. |
| P-03 | Pérdida parcial o total de solicitudes debido a la ausencia de un sistema centralizado de registro. |
| P-04 | Falta de claridad sobre los responsables encargados de revisar, aprobar o rechazar las solicitudes. |
| P-05 | Retrasos frecuentes en los tiempos de respuesta y aprobación de permisos o vacaciones. |
| P-06 | Ausencia de un historial digital que permita realizar seguimiento y consulta de solicitudes anteriores. |
| P-07 | Generación de estrés, incertidumbre y desmotivación en los empleados debido a la falta de información sobre el estado de sus solicitudes. |
| P-08 | Falta de centralización de la información relacionada con permisos, vacaciones y solicitudes administrativas. |
| P-09 | Dificultad para consultar el estado actualizado de las solicitudes en tiempo real. |
| P-10 | Falta de notificaciones o confirmaciones automáticas que informen la recepción y avance de una solicitud. |
| P-11 | Baja eficiencia en la comunicación entre los empleados y el área de Talento Humano. |
| P-12 | Riesgo de inconsistencias o errores en la gestión manual de solicitudes y aprobaciones. |
