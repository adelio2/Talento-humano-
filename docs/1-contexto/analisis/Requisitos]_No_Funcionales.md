# Requisitos No Funcionales

| Código | Categoría | Requerimiento No Funcional |
|---|---|---|
| RNF-01 | Integridad y trazabilidad | El sistema debe garantizar la integridad, trazabilidad e inmutabilidad de la información registrada en cada solicitud y acción realizada. |
| RNF-02 | Usabilidad | El sistema debe contar con una interfaz intuitiva, organizada y fácil de usar para empleados y personal de Talento Humano. |
| RNF-03 | Rendimiento | El sistema debe responder a las solicitudes de consulta y registro en un tiempo máximo de 3 segundos bajo condiciones normales de uso. |
| RNF-04 | Disponibilidad | El sistema debe mantenerse disponible durante la jornada laboral institucional para garantizar el acceso continuo de los usuarios. |
| RNF-05 | Seguridad | El sistema debe implementar autenticación segura mediante usuario y contraseña para controlar el acceso a la plataforma. |
| RNF-06 | Seguridad | El sistema debe gestionar permisos y restricciones de acceso según el rol asignado a cada usuario. |
| RNF-07 | Confiabilidad | El sistema debe garantizar que las solicitudes registradas no se pierdan, eliminen accidentalmente o se dupliquen durante el proceso. |
| RNF-08 | Escalabilidad | El sistema debe soportar el incremento de usuarios y solicitudes sin afectar significativamente el rendimiento de la plataforma. |
| RNF-09 | Compatibilidad | El sistema debe ser compatible con los principales navegadores web y dispositivos móviles. |
| RNF-10 | Mantenibilidad | El sistema debe contar con documentación técnica y estructurada para facilitar el mantenimiento y futuras mejoras. |
| RNF-11 | Disponibilidad | El sistema debe realizar copias de seguridad periódicas para proteger la información almacenada. |
| RNF-12 | Comunicación | El sistema debe enviar notificaciones automáticas de manera confiable y oportuna ante cambios en las solicitudes. |
| RNF-13 | Accesibilidad | El sistema debe presentar información clara y legible para facilitar la navegación y comprensión de los usuarios. |
| RNF-14 | Centralización | El sistema debe centralizar toda la información relacionada con solicitudes laborales en una única plataforma digital institucional. |
