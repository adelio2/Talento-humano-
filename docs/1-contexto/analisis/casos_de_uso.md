# Casos de Uso

| Código | Caso de Uso | Actor Principal | Descripción |
|---|---|---|---|
| CU-01 | Registrar solicitud de permiso laboral | Empleado | Permite al empleado registrar solicitudes de permisos laborales mediante la plataforma institucional. |
| CU-02 | Solicitar vacaciones | Empleado | Permite al empleado registrar solicitudes de vacaciones dentro del sistema de gestión de Talento Humano. |
| CU-03 | Registrar incapacidad médica | Empleado | Permite al empleado registrar incapacidades médicas adjuntando documentos de soporte requeridos. |
| CU-04 | Solicitar certificación laboral | Empleado | Permite al empleado solicitar certificaciones laborales de manera formal desde la plataforma institucional. |
| CU-05 | Consultar estado de solicitud | Empleado | Permite al empleado visualizar el estado actualizado y seguimiento de las solicitudes registradas. |
| CU-06 | Consultar historial de solicitudes | Empleado | Permite al empleado consultar el historial de solicitudes realizadas y sus respuestas registradas. |
| CU-07 | Consultar responsable asignado | Empleado | Permite al empleado identificar el responsable encargado de gestionar su solicitud. |
| CU-08 | Recibir notificaciones automáticas | Empleado | Permite al empleado recibir notificaciones automáticas relacionadas con cambios de estado o respuestas de sus solicitudes. |
| CU-09 | Visualizar solicitudes registradas | Responsable de Talento Humano | Permite consultar y visualizar todas las solicitudes registradas en el sistema. |
| CU-10 | Aprobar solicitudes | Responsable de Talento Humano | Permite aprobar solicitudes realizadas por los empleados según el proceso establecido. |
| CU-11 | Rechazar solicitudes | Responsable de Talento Humano | Permite rechazar solicitudes registrando observaciones o motivos del rechazo. |
| CU-12 | Cambiar estado de solicitud | Responsable de Talento Humano | Permite actualizar el estado de una solicitud durante el proceso de gestión. |
| CU-13 | Consultar historial y trazabilidad | Responsable de Talento Humano | Permite consultar el historial de acciones y cambios realizados sobre las solicitudes. |
| CU-14 | Generar reportes administrativos | Responsable de Talento Humano | Permite generar reportes relacionados con solicitudes, estados y tiempos de respuesta. |
| CU-15 | Gestionar responsables del proceso | Responsable de Talento Humano | Permite asignar responsables para la revisión y seguimiento de las solicitudes registradas. |
| CU-16 | Validar información de solicitudes | Sistema | Permite validar campos obligatorios y documentos adjuntos antes de registrar una solicitud. |
| CU-17 | Generar número de radicado | Sistema | Permite generar automáticamente un identificador único para cada solicitud registrada. |
| CU-18 | Registrar trazabilidad de acciones | Sistema | Permite almacenar automáticamente las acciones y cambios realizados sobre cada solicitud. |
