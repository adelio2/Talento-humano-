# Requisitos No Funcionales

| Código | Categoría | Requerimiento No Funcional |
|---|---|---|
| RNF-01 | Integridad y trazabilidad | El sistema debe garantizar la integridad, trazabilidad e inmutabilidad de la información registrada en cada solicitud y acción realizada dentro de la plataforma. |
| RNF-02 | Usabilidad | El sistema debe contar con una interfaz intuitiva, organizada y fácil de usar para empleados y responsables de Talento Humano. |
| RNF-03 | Rendimiento | El sistema debe responder a las solicitudes de consulta, registro y actualización en un tiempo máximo de 3 segundos bajo condiciones normales de operación. |
| RNF-04 | Disponibilidad | El sistema debe mantenerse disponible durante la jornada laboral institucional para garantizar el acceso continuo de los usuarios autorizados. |
| RNF-05 | Seguridad | El sistema debe implementar mecanismos de autenticación segura mediante usuario y contraseña para controlar el acceso a la plataforma. |
| RNF-06 | Seguridad | El sistema debe gestionar permisos y restricciones de acceso de acuerdo con el rol asignado a cada usuario. |
| RNF-07 | Confiabilidad | El sistema debe garantizar que las solicitudes registradas no se pierdan, eliminen accidentalmente o se dupliquen durante el proceso de gestión. |
| RNF-08 | Escalabilidad | El sistema debe soportar el incremento de usuarios y solicitudes sin afectar significativamente el rendimiento de la plataforma. |
| RNF-09 | Compatibilidad | El sistema debe ser compatible con los principales navegadores web y dispositivos móviles utilizados por los usuarios. |
| RNF-10 | Mantenibilidad | El sistema debe contar con documentación técnica estructurada que facilite el mantenimiento y futuras mejoras del software. |
| RNF-11 | Respaldo de información | El sistema debe realizar copias de seguridad periódicas para proteger la información almacenada y garantizar su recuperación en caso de fallos. |
| RNF-12 | Comunicación | El sistema debe enviar notificaciones automáticas de manera confiable y oportuna ante cambios en el estado de las solicitudes. |
| RNF-13 | Accesibilidad | El sistema debe presentar información clara, legible y organizada para facilitar la navegación y comprensión de los usuarios. |
| RNF-14 | Centralización | El sistema debe centralizar toda la información relacionada con solicitudes laborales en una única plataforma digital institucional. |
